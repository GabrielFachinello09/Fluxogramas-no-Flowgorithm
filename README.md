# Exercícios S1_R2-AT1_LOPAL
Exercício 1

O algoritmo inicia solicitando ao usuário que digite seu nome. Após a leitura dessa informação, o sistema vai realizar apenas uma ação de exibição, apresentando uma mensagem de saudação ao usuário “Olá, seja bem vindo, [nome informado]!”

Exercício 2


O fluxograma começa com a leitura de dois números informados pelo usuário. Em seguida, o algoritmo processa esses valores realizando o cálculo da soma entre eles. Por fim, o resultado obtido é exibido na tela.

Exercício 3

O algoritmo solicita ao usuário que informe sua idade. Após a leitura, é feita uma verificação condicional: caso a idade seja maior ou igual a 18 anos, o sistema exibe a mensagem “Maior de idade”. Caso contrário, apresenta “Menor de idade”.
